# PRISM-IR Examples

This directory contains worked PRISM-IR examples. Each file demonstrates
the round-trip principle: a natural-language description and a typed IR
that mean the same thing.

## Index

| File | Domain | Notes |
|------|--------|-------|
| `claims-processing.prism.md` | Insurance | Workflow shown in the LinkedIn announcement image. Validate policy, branch on activity status, branch on amount threshold, converge on customer notification and audit. |

More examples will be added as the round-trip harness expands. See
[`PATTERNS.md`](../PATTERNS.md) for the van der Aalst pattern coverage
table once it is published.
